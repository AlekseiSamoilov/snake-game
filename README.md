## üéÆ About the Project

"Snake: Strategic Game" is a modern interpretation of the classic Snake game with unique mechanics and strategic gameplay. Unlike the traditional version, our game focuses on managing a long snake and strategically using the game field.

### üåü Features
- üçé Various types of food with unique effects
- ‚úÇÔ∏è Ability to bite off part of the snake to create obstacles
- üèîÔ∏è Petrified snake parts become obstacles on the field
- üß† Emphasis on strategy rather than speed
- üåê Online leaderboard

## üõ†Ô∏è Technologies
- Frontend: React, TypeScript
- Backend: Nest.js
- Database: MongoDB

## üöÄ Installation and Setup

1. Clone the repository:
   ```
   git clone https://github.com/AlekseiSamoilov/snake-game.git
   ```
2. Navigate to the project directory:
   ```
   cd snake-game
   ```
3. Install dependencies using yarn:
   ```
   yarn install
   ```
4. Start the frontend:
   ```
   yarn start:frontend
   ```
5. In a separate terminal, start the backend:
   ```
   yarn start:backend
   ```
